About the JBPB

The Jamal Bourne Phone Book (JBPB) is a simple phone book built using the Java Framework, Spring MVC. It also includes the following technologies:

H2 Database

  • Very Fast and Open Source
  • In Memory Database
  • Browser based Console application
  • Small footprint

Bootstrap

  • Popular Front-End Open Source Toolkit
  • Responsive Grid System
  • Pre Built Components

Embeded Server

  • Uses an Embeded Tomcat Server
  • Ratio between the server and the application is 1-to-1
